# DDD Cargo App
Implementation of the Cargo Shipping System described in "Domain Driven Design - Tackling Complexity in the Heart of Software" chapter 7.

## Setup
```bash
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
```

## Run tests
```bash
pytest
```
